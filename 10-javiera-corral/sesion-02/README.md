MAQUINA DIFERENCIAL (the difference engine) 1822
--
“LA PRIMERA COMPUTADORA DEL MUNDO”

Babbage diseñó esta máquina para sistematizar el
cálculo y la tabulación de funciones polinómicas, tareas
que hasta entonces se realizaban manualmente y eran
propensas al error. Se trataba de un dispositivo mecánico
de gran escala, compuesto íntegramente por engranajes
que servía para resolver ecuaciones e imprimir los
resultados en forma de tablas matemáticas.

MÁQUINA ANALÍTICA 1834 -1837
--
<img width="442" height="317" alt="image" src="https://github.com/user-attachments/assets/c8021384-e7d4-4102-a316-4ef96547ecfd" />

La Máquina Analítica fue diseñada para realizar
cualquier tipo de cálculo establecido anteriormente y
para tener una capacidad de análisis aún mayor que la
Máquina Diferencial original. Se considera la primera
máquina de cálculo completamente automática.

El Molino - The Mill: La CPU o Procesador donde se hacen las operaciones.
El Almacén - The Store: La Memoria donde se guardan los números y datos.
Las Tarjetas perforadas - Punch card system : El Software entrada de datos e instrucciones.

Telar De Jacquard 1801
--

![Uploading image.png…]()

• Primer sistema de almacenamiento de información binaria.

• El telar usaba un sistema de tarjetas perforadas para
lograr diseños o dibujos complejos y automatizados.

• Si había agujero: La varilla pasaba, el gancho se activaba y
levantaba el hilo (podemos llamarlo un 1 o “Encendido").

• Si NO había agujero: La varilla chocaba contra la tarjeta, el
gancho no se movía y el hilo se quedaba abajo (podemos
llamarlo un 0 o "Apagado").

El Telar de Jacquardintrodujo tres conceptos fundamentales de la computación moderna:  
--

1. LA SEPARACIÓN ENTRE HARDWARE Y SOFTWARE
Hardware: El telar en sí (la máquina física, los engranajes, los hilos).
Software: Las tarjetas perforadas.

Antes de Jacquard, si queríamos que una máquina hiciera algo diferente,
teníamos que reconstruirla o cambiar sus piezas. Con Jacquard, la
máquina no cambia; solo cambias las tarjetas (el programa) y obtienes un
resultado totalmente distinto.

2. EL SISTEMA BINARIO
Aunque Jacquard no pensaba en matemáticas, creó un sistema de lógica
binaria. La ausencia o presencia de materia (agujero o cartón) determinaba
una acción. Es el antepasado directo del transistor moderno (paso de
corriente vs. no paso de corriente)

3. EL PÍXEL Y LA IMAGEN RASTERIZADA
Podemos entender el tejido como una forma primigenia de imagen de mapa
de bits: la tela constituye la rejilla (grid) de soporte, donde cada intersección
entre trama y urdimbre funciona como un píxel individual. Bajo esta lógica, la
tarjeta perforada opera como el archivo digital (el .jpg), enviando las
instrucciones precisas sobre qué puntos activar para componer la imagen
visual.
