## Loop
¿Los loops son infinitos?  
### Bucle definición:
1. Rizo de cabello de forma helicoidal.
2. Objeto ciya forma recuerda la del bucle.
3. Proceso que se repite indefinidamente.
4. Infórmatica

### Loop.
Es una estructura de control que permite ejecutar un bloque de instruccipnes de manera repetida mientras se cumpla una condición especifica o hasta que se alcance un estado determinado.   

### While 3 elementos
los bucles while son útiles para repetir instrucciones mientras una condición sea verdadera. Son como sentencias if que se repite.
(condición booleana) {si es true ejecuta este código en BUCLE}   

### For 4 elementos 
Una forms de repetir un bloque de código cuando se conoce el número de iteraciones. Los bucles `for` son útiles para repetir instrucciones un número determinado de veces.
Son una especie de SHORTCUT para hacer loops y siempre tienen 4 elementos:  
1. inicialización de una variable
2. Condición booleana (V-F)
3. Actualización (incrementación o decrementación)
4. Lo que queremos que pase cuando la condición sea TRUE
