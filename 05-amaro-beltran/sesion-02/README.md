# Breve historia de la computación  
## Arquitectura del pensamiento  
1882-**Charles Babbage: Máquina diferencial:** Diseñó esta máquina para sistematizar cálculo y tabulación de funciones polinomicas.  
1991 el Museo de las Ciencias de Londres la construyó siguiendo los planos originales.  
1834-37 **Máquina analítica de Charles Babbage** \(hace calcúlos)  
* El molino - CPU - donde se hacen las operaciones
* El almacén - memoria - guardan datos y números
* Tarjetas perforadas - software - entrada de datos e instrucciones

1801 **Telar de Jacquard - Joseph Jacquard**: primer sistema de almacenamiento de información binaría, usaba tarjetas perforadas, instrucciones si y no \(0 y 1)
  1. Separación entre Hardware y Software \(telar y tarjetas)
  2. Sistema binario \(0 y 1)
  3. Pixel

**Ada Lovelace** \(1815-1852): entendío el potencial de las tarjetas perforadas, progrmación - manipular simbolos, creo el primer algoritmo, primera programadora  
**Máquina de Turing - Alan Turing** 1936: experimento mental, da igual el Hardware importan las indicaciones.  

# Despertar Visual  
## Artistas y algoritmos  
1960 comienzan esfuerzos por crear arte con la computadora en medio de una escena artistica efervecente.  
1968 **Exposición Cybernetic Serendipity Londres**: busca mostrar como se pueden usar las tecnologias para extender la creatividad  
1969 - **Argentina Arte y Cibernetica** organizado por **Jorge Glusberg**.  
**John y James Whitney**: pioneros del Computer Graphics Animation, diseño parametrico, movimiento armonico, usaban computadores originalmente diseñados para la guerra, primera animación por computador.  
**Ivan Sutherland** 1963: creo el primer programa que permite dibujar en una pantalla con un lapiz optico.  
**Douglas Engelbart** 1968: Bitmaping, mouse.
