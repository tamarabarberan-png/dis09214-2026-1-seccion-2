# Pensamiento computacional  
## 4 pilares  
descomposición
reconocimiento de patrones
abstracción
algoritmos  
## Descomposición  
Consiste en tomar un problema grande y complejo, y separarlo en partes más pequeñas y manejables. Se traduce en el uso de funciones propias.  

## Reconocimiento de patrones  
Observar tendencias dentro de un problema, si algo se repite se puede automatizar en bucle (for).  

## Abstracción  
Filtrar información innecesaria. Se traduce en el uso de variables.  

## Algoritmos  
Diseño de serie de reglas ordenadas para resolver un problema. Se traduce en el diagrama de flujo y condicionales.  

## Tipos de interacción  
* interacción discreta: cuando ocurre un evento específico (click) y el sistema responde con una acción única.
* interacción continua: cuando el sistema reacciona constantemente al movimiento o estado del usuario, sin necesidad de hacer algo específico (clic).
